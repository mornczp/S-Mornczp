# tp.file.folder()
检索文件的文件夹名称。
```php
tp.file.folder(relative: boolean = false)
```

```php
//文件文件夹（文件夹）
Banner
//具有相对路径的文件文件夹（路径/目标/文件夹）
content/04-Mornczp/Mornczp载体/Obsidian/Obsidian_Plugins/外观&界面拓展/Banner
//显示链接
[[Banner]]  
```
